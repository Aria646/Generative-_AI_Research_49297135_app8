# Generative AI Research Project

## Overview

This repository contains all materials related to the research project **“Using Generative AI Tools – Boon or Bane”**. It is designed to support effective data management, version control, and collaborative research practices using GitHub.

The repository organises different types of research data, including literature sources, quantitative and qualitative datasets, analysis materials, and project reports, in a structured and systematic manner.

---

## Project Structure

- `Literature_Review/`  
  Contains journal articles, academic papers, and reference materials used to support the research background.

- `Quantitative_Analysis/`  
  Includes survey datasets (CSV files), survey questions, and analysis scripts (e.g., Python) used for statistical analysis.

- `Qualitative_Analysis/`  
  Contains interview transcripts and qualitative research materials used for thematic analysis and interpretation.

- `Drafts_and_Reports/`  
  Includes research proposals, draft papers, and final reports documenting the progress and outcomes of the project.

- `Additional_Materials/`  
  Stores images and supporting materials related to the project.

---

## How to Navigate

Each top-level folder corresponds to a specific research activity. Users can navigate the repository by selecting the relevant folder to access literature, datasets, transcripts, analysis files, and reports.

Files are organised to ensure clarity, ease of access, and efficient collaboration among team members.

---

## Data Management Practices

This repository follows key data management principles:

- Structured organisation of files based on data type and research activity  
- Separation of raw data, analysis, and reports  
- Use of version control to track changes and maintain data integrity  
- Consideration of ethical requirements when handling sensitive data  

---

## Collaboration Guidelines

To ensure effective teamwork and consistency:

- Use clear and consistent file naming conventions  
- Write meaningful commit messages (e.g., “Added survey data and initial analysis script”)  
- Review files before uploading to maintain organisation  
- Use branches and pull requests when collaborating on shared work  

---

## Ethical Considerations

Some research materials, particularly interview transcripts and participant-related data, may contain sensitive information. These files should be handled carefully, with restricted access where necessary, to ensure privacy, confidentiality, and compliance with ethical research standards.
